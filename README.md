# WeatherProject
With CoreData

Description
-----------
The app shows the weather when you enter a city name and through coordinates using AppleMap and GoogleMap. 
And the map selection is done through Firebase. You will also find advertising that is made with Google Ads(rewarded, interstitial, baner). 
The app has various sounds, videos and animations. There are also world news in the magazine "The Wall Street Journal". 
And the history of weather requests stored in the database using CoreData. 
The app is localized in three languages such as English, Russian and Polish.

APLIED
------
Applied: UIKit, AVKit, MapKit, Xib, Networking, JSON, Localization, DateFormatter, TableView and 
Scroll View, Working with files, Cocoapods(Alamofire, ActivityIndicator, GoogleMaps, lottie, ...), 
Farebase(RemoteConfig, GoogleAds), CoreData, GCD, RxSwift(tableView, UIBatton), MVC.
